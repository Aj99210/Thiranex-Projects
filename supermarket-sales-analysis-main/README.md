# 🛒 Supermarket Sales Analysis

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on supermarket sales data to extract meaningful business insights such as sales trends, customer behavior, and product performance.

---
## 🚀 Objective
To practice real-world data analysis using Python and extract actionable insights from raw business data.

---

## 🛠 Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

---

## 📂 Dataset
Dataset taken from Kaggle (Supermarket Sales Dataset)

---

## 📊 Key Analysis Performed
- Data cleaning and preprocessing
- Sales trend analysis across branches
- Product category performance analysis
- Customer purchasing behavior analysis
- Revenue and profit insights using visualizations

---

## 📈 Key Insights
- Identified top-selling product categories contributing highest revenue
- Found that specific branches performed significantly better in sales
- Observed peak sales hours based on customer purchasing behavior
- Discovered strong relationship between product type and revenue generation
- Highlighted customer buying patterns across different demographics

---

## 📁 Files in this Repository
- Supermarket_Sales.ipynb → Main analysis notebook
- Summary_Report.pdf → Final insights report
- report.pdf (summary report of findings)

## 📊 Results
- Improved understanding of sales trends and customer behavior
- Generated actionable insights for business decision-making
- Created visualizations to support data-driven conclusions
